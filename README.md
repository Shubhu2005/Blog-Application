# React Blog Application

This is a simple Blog Application built using **React.js** as part of my internship task.  
The app allows users to **add, edit, delete, and search blogs**.  
Blogs are also saved in **localStorage**, so they remain even after page refresh.

---

## Features
- Add a new blog  
- Edit an existing blog  
- Delete a blog  
- Search blogs  
- Show created date  
- Dark mode toggle  
- LocalStorage support  
- Clean and simple UI
